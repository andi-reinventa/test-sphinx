/upload

POST /api/upload (Ejemplo)

Gestiona la carga de archivos a un contenedor de Azure Blob Storage.

Cabeceras Requeridas:

  • container-name (string) – El nombre del contenedor donde se guardará el archivo.

  • azure-token (string) – El token SAS para la autenticación con Azure Blob Storage.

Parámetros del Formulario:

  • file (archivo) – El archivo a subir.

Advertencia

Asegúrate de que el token SAS y el nombre del contenedor son válidos para evitar errores de autenticación.

Respuestas:

Códigos de Status:

  • 401 Unauthorized – No autorizado - Si el usuario no está autenticado.

  • 400 Bad Request – Solicitud Incorrecta - Si las cabeceras requeridas no están presentes.

  • 303 See Other – Vea Otros - Redirección tras una carga exitosa.

  • 500 Internal Server Error – Error Interno del Servidor - Si ocurre un error durante la carga del archivo.

Ejemplo de uso:

 1// upload-example.js
 2const axios = require('axios').default;
 3const fs = require('fs');
 4const FormData = require('form-data');
 5
 6async function uploadFile(filePath, containerName, azureToken) {
 7    try {
 8        // Create a form data object and append the file to upload
 9        const formData = new FormData();
10        formData.append('file', fs.createReadStream(filePath));
11
12        // Set the headers for the request, including the container-name and azure-token
13        const headers = {
14            ...formData.getHeaders(),
15            'container-name': containerName,
16            'azure-token': azureToken,
17        };
18
19        // Make the POST request to the server to upload the file
20        const response = await axios.post('http://localhost:3000/api/upload', formData, { headers });
21
22        // Handle the response from the server
23        console.log('File uploaded successfully:', response.data);
24    } catch (error) {
25        // Handle any errors that occur during the upload process
26        if (error.response) {
27            console.error('Failed to upload file:', error.response.status, error.response.data);
28        } else {
29            console.error('Error uploading file:', error.message);
30        }
31    }
32}
33
34// Example usage of the uploadFile function
35// Replace 'path/to/file.txt' with the actual file path
36// Replace 'your-container-name' with your actual Azure container name
37// Replace 'your-azure-sas-token' with your actual Azure SAS token
38uploadFile('path/to/file.txt', 'your-container-name', 'your-azure-sas-token');

Nota

Este ejemplo utiliza el middleware auth.js para asegurar que el usuario esté autenticado antes de permitir la carga del archivo.
