Diseño

Podemos realizar el número de formularios que deseemos, con la cantidad de prompts nativos disponibles en la interfaz que necesitemos, simplemente agregándolos al listado de «Prompts seleccionados».

También puede solicitar la creación de los prompts que precise el servicio concreto de forma rápida y sencilla, contactando con nosotros.

Para localizar los prompt correspondientes, la plataforma dispone de un buscador con el que localizar a través de palabras clave.

Del mismo modo, cada una de las definiciones o funcionalidad entregada por cada prompt del sistema, se define y explica manteniendo el puntero del ratón sobre el título del mismo.
